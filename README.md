# json-to-csv

Простой скрипт на Python для конвертации JSON-файлов в CSV-файлы.

## Описание

Этот проект предоставляет инструмент командной строки для автоматической конвертации всех JSON-файлов в текущей директории в CSV-файлы. Поддерживает как массивы объектов JSON, так и одиночные объекты.

## Установка

1. Убедитесь, что у вас установлен Python 3.13 или выше.
2. Клонируйте репозиторий или скачайте файлы проекта.
3. Установите зависимости с помощью uv:

   ```
   uv sync
   ```

## Использование

Запустите скрипт из командной строки:

```
python json_to_csv.py
```

Это автоматически найдет все файлы с расширением `.json` в текущей директории и конвертирует их в соответствующие `.csv` файлы.

### Примеры

1. Конвертация всех JSON-файлов в директории:

   ```
   python json_to_csv.py
   ```

   Если в директории есть файлы `data.json`, `input.json` и т.д., будут созданы `data.csv`, `input.csv` и т.д.

### Поддерживаемые форматы JSON

- **Массив объектов**: `[{"key1": "value1", "key2": "value2"}, ...]`
- **Одиночный объект**: `{"key1": "value1", "key2": "value2"}`

Для массивов объектов ключи первого объекта используются как заголовки CSV.

### Формат CSV

Выходной CSV-файл использует точку с запятой (`;`) в качестве разделителя полей для совместимости с некоторыми европейскими локалями и приложениями, такими как Excel в некоторых регионах.
