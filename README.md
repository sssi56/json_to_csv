# json-to-csv

Простой скрипт на Python для конвертации JSON-файлов в CSV-файлы.

## Описание

Этот проект предоставляет инструмент командной строки для преобразования данных из формата JSON в формат CSV. Скрипт поддерживает как массивы объектов JSON, так и одиночные объекты.

## Установка

1. Убедитесь, что у вас установлен Python 3.13 или выше.
2. Клонируйте репозиторий или скачайте файлы проекта.
3. Установите зависимости (если есть) с помощью uv:

   ```
   uv sync
   ```

## Использование

Запустите скрипт из командной строки:

```
python main.py <путь_к_json_файлу> [--output <путь_к_csv_файлу>]
```

### Примеры

1. Конвертация JSON-файла с автоматическим именем выходного файла:

   ```
   python main.py data.json
   ```

   Это создаст файл `data.csv` в той же директории.

2. Конвертация с указанием выходного файла:

   ```
   python main.py input.json --output output.csv
   ```

### Поддерживаемые форматы JSON

- **Массив объектов**: `[{"key1": "value1", "key2": "value2"}, ...]`
- **Одиночный объект**: `{"key1": "value1", "key2": "value2"}`

Для массивов объектов ключи первого объекта используются как заголовки CSV.

### Формат CSV

Выходной CSV-файл использует точку с запятой (`;`) в качестве разделителя полей для совместимости с некоторыми европейскими локалями и приложениями, такими как Excel в некоторых регионах.

## Требования

- Python >= 3.13
